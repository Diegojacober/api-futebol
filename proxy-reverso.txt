Como configurar o proxy reverso para um servidor:


sudo apt install nginx

sudo service nginx status -> ver status

sudo service nginx start -> iniciar servidor

sudo nano /etc/nginx/sites-enabled/api-node -> escreve o arquivo de configuração do nginx

cd /etc/nginx/sites-enabled/ - sudo rm -rf default -> apaga a página padrão do nginx


sudo nginx -t -> só sei que tem que fazer 

sudo service nginx restart -> reinicia nginx
