Configurar um servidor para conexões ssh:

Primeiro encontra o endereço de ip do mesmo

faz um teste de ping pra ver se realmente esta conseguindo acessar

sudo apt install ssh -> instalando um servidor ssh

sudo nano /etc/ssh/sshd_config 

passwordAuthentication no -> bloqueando acesso por senha
maxAuthTries -> aumentar 

sudo systemctl restart ssh -> caso o systemctl não funcione no wsl, podemos habilita-lo: https://devblogs.microsoft.com/commandline/systemd-support-is-now-available-in-wsl/

ou usar: sudo service ssh restart

ssh-keygen -f ~/.ssh/nomeDachavve -t rsa -b 4096 -> gerar a chave ssh no pc que vai acessar o servidor
	

copiar a chave ssh publica

no servidor abrir o arquivo: nano ~/.ssh/authorized_keys -> colar a chave ssh

se for windows toda vez que reiniciar o pc é necessário digitar os seguintes comandos:

eval $(ssh-agent)
ssh-add ~/.ssh/id_rsa
